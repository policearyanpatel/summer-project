# Task Manager Pro

## About the Project

Task Manager Pro is a desktop-based task management application developed using Python and CustomTkinter. The application helps users organize and manage their daily tasks through a simple graphical interface.

The project allows users to create tasks, assign priorities, set due dates, track task status, and monitor productivity. Task data is stored locally using JSON, ensuring that tasks remain available even after the application is closed and reopened.

This project was developed to strengthen my understanding of Python programming, GUI development, file handling, and application design.

---

## Features

### Task Management

* Add new tasks
* Update existing tasks
* Delete tasks
* Unique task ID generation
* Task notes support

### Task Organization

* Categories for better organization
* Priority levels (High, Medium, Low)
* Status tracking (Pending, In Progress, Completed)
* Due date management

### Productivity Tracking

* Total task count
* Completed task count
* Pending task count
* Productivity percentage
* Achievement tracking

### Search and Monitoring

* Search tasks by title or task ID
* View overdue tasks

### Data Storage

* Automatic task saving using JSON
* Automatic task loading when the application starts

### Reports

* Export task information to a text report

## How the Application Works

1. Launch the application.
2. Add a task by entering the required details.
3. Assign a category, priority level, status, and due date.
4. Update task information whenever required.
5. Search tasks using the search feature.
6. Monitor productivity through the dashboard.
7. View overdue tasks when needed.
8. Export task information to a report file.

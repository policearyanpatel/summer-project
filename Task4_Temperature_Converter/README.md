# Temperature Converter Pro+

## About the Project

Temperature Converter Pro+ is a desktop application developed using Python and CustomTkinter. The application allows users to convert temperature values between Celsius, Fahrenheit, and Kelvin through a simple and interactive graphical interface.

The goal of this project was to build a practical desktop application while improving skills in Python GUI development, event handling, data processing, and user interface design.

---

## Features

* Convert temperatures between Celsius, Fahrenheit, and Kelvin
* Simple and user-friendly graphical interface
* Conversion history tracking
* Search history section
* Statistics panel displaying:

  * Total conversions
  * Highest temperature
  * Lowest temperature
  * Average temperature
* Temperature status indicator
* Temperature meter visualization
* Preset temperature buttons for quick testing
* Swap unit functionality
* Copy result to clipboard
* Export conversion history to a text file
* Live clock display
* Input validation and error handling

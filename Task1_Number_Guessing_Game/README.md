# Number Guess Master Pro

## About the Project

Number Guess Master Pro is a Python-based number guessing game developed using CustomTkinter. The player has to guess a randomly generated number within a limited number of attempts. The game provides hints, score tracking, ranks, and stores the highest score achieved by the player.

This project was created to practice Python programming concepts such as GUI development, object-oriented programming, file handling, and input validation.

---

## Features

* Multiple difficulty levels (Easy, Medium, Hard)
* Random number generation
* Limited attempts based on difficulty
* Score calculation system
* Player ranking system
* High score storage using a text file
* Guess history tracking
* Even/Odd hints
* Range-based hints
* Progress bar for attempts
* Input validation
* Winning and Game Over messages
* Modern GUI using CustomTkinter

## How to Run

1. Install CustomTkinter

```bash
pip install customtkinter
```

2. Open the project folder

```bash
cd Task1_Number_Guessing_Game
```

3. Run the application

```bash
python guessing_game_gui.py
```
